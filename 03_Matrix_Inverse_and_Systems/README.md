# 第 03 阶段：逆变换与线性方程组 (Matrix Inverse & Systems)

在第 2 阶段中，我们学习了如何通过矩阵乘法 $Ax = b$ 对空间进行正向变换（从 $x$ 变到 $b$）。
本章我们将逆向思考：**如果知道变换后的结果 $b$，我们如何反向退回到原向量 $x$？这就是解线性方程组和求逆矩阵的本质。**

---

## 📐 1. 数学核心与几何直觉

### 1.1 逆变换与逆矩阵 (Inverse Matrix)
*   **几何本质**：如果矩阵 $A$ 对空间做了一次特定的“动作”（例如顺时针旋转 30°），那么它的逆矩阵 $A^{-1}$ 就是**反向撤销**这一动作的“解药”（逆时针旋转 30°）。
*   **计算公式**：
    $$A^{-1}A = I \quad (\text{单位矩阵，相当于不进行任何动作})$$
*   **无解的情况**：如果 $\det(A) = 0$，表示变换将空间压缩降维了（比如 3D 变 2D 投影）。由于丢失了整整一个维度的数据，我们**不可能**通过任何变换把一条线“复原”回原来的立体空间。因此，降维矩阵**没有逆矩阵**，对应的方程组可能无解或有无数解。

### 1.2 列空间与秩 (Column Space & Rank)
*   **列空间 (Column Space)**：矩阵所有列向量的 Span（张成空间）。由于矩阵的列是基向量变换后的位置，因此列空间就是**变换后新空间的所有可能取值的集合**。
*   **秩 (Rank)**：变换后空间的实际维度。
    *   如果一个 $3 \times 3$ 矩阵的列向量全部落在同一个 2D 平面上，那么它的秩是 $2$。
    *   **满秩 (Full Rank)**：秩等于矩阵的列数。此时没有维度丢失，变换是可逆的。

### 1.3 投影与最小二乘法 (Projection & Least Squares)
当我们要求解 $Ax = b$ 时，如果 $b$ 不在矩阵 $A$ 的列空间中（即 $b$ 处于变换所能到达的空间之外），方程组就**无解**。
在实际应用中，我们的数据往往充满噪音，导致 $b$ 偏离了完美的列空间。此时，我们的最佳选择是：**寻找列空间中距离 $b$ 最近的那个向量。**
*   **几何结论**：距离最近的向量就是 $b$ 在 $A$ 列空间上的**正交投影**。
*   根据正交性质，我们可以推导出**正规方程 (Normal Equation)**：
    $$A^T A x = A^T b \quad \Rightarrow \quad x = (A^T A)^{-1} A^T b$$
    这就是著名的**最小二乘法 (Least Squares)**。

## 🤖 2. AI 映射：线性回归与自编码器

### 2.1 线性回归 (Linear Regression) 的几何内涵
我们拟合一条直线 $y = wx + b$ 时，如果把所有训练样本拼成一个巨大的矩阵方程 $Aw = y$，这个方程通常由于样本噪音而无解。
*   **几何视角**：我们不是在求一个完美的解，而是将目标值向量 $y$ **投影**到特征空间（由特征列向量张成）上。
*   通过投影，我们得到了最佳的权重系数 $w$。

### 2.2 自编码器 (Autoencoder) 与信息重构
自编码器（Autoencoder）是一种无监督学习模型，其结构为：
$$\text{输入 } x \xrightarrow{\text{Encoder}} \text{低维瓶颈空间 } z \xrightarrow{\text{Decoder}} \text{重构 } \hat{x}$$
*   **空间变换视角**：
    *   **Encoder**：执行的是降维空间映射（类似于有损压缩，行列式接近 0 的变换）。
    *   **Decoder**：试图在降维后的限制空间中，重构出与原始输入 $x$ 距离最近的向量。这本质上是在寻找高维流形空间中的最优投影！
